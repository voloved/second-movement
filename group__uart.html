◆ watch_enable_uart()

void watch_enable_uart ( const uint16_t tx_pin,
const uint16_t rx_pin,
uint32_t baud )

Initializes the UART.

Parameters
tx_pinThe pin the watch will use to transmit, or 0 for a receive-only UART. If specified, must be either HAL_GPIO_A2_pin() or HAL_GPIO_A4_pin().
rx_pinThe pin the watch will use to receive, or 0 for a transmit-only UART. If specified, must be A1, A2, A3 or A4 (pin A0 cannot receive UART data).
baudThe baud rate for the UART. A typical value is 19200.
