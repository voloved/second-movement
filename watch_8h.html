void irq_handler_system ( void )

Interrupt handler for the SYSTEM interrupt, which handles MCLK, OSC32KCTRL, OSCCTRL, PAC, PM and SUPC.

We use this to detect when the system voltage has dipped below 2.6V, which means it's time to disable the voltage regulator's high-efficiency mode, which only works down to 2.5V.
